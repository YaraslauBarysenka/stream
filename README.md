# Микросервис: интеграция с API стримов

> **Для компании RED** | статическое ревью кода

---

## Важно

Проект **не собирается и не запускается** (отсутствует часть кода).  
Предназначен только для оценки архитектуры и стиля.

---

## О проекте

Интеграция с внешним поставщиком стримов.  
Реализован приблизительно за **3 недели**.

---

## Основная логика
1. Получаем данные по REST (FeignClient), используя Scheduled Job;
2. Запрашиваем Events за временной интервал и синхронизируем с локальной БД;
3. Все изменения в Events пишем в Kafka топик, через отдельную change таблицу, используя Transaction Outbox pattern.
