# Микросервис: интеграция с API стримов

> **Для компании RED | статическое ревью кода**

Проект не собирается и не запускается (отсустствуте часть кода).

Реализован приблизительно за 3 недели. Интеграция с внешним поставщиком стримов.

Основная логика: получам по ресту провайдера (Feign client) Events по временному интервалу и синхронизируем с локальной базой данных.
Все необходимые Event изменения пишем в Кафка топик, через отдельную таблицу (Transaction Outbox pattern).
